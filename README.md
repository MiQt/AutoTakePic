如何使用它？

第一步：..\YourProject\build.gradle

	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
第二步：..\YourProject\app\build.gradle

	dependencies {
	        compile 'com.github.MiQt:AutoTakePic:eede92a542'
	}
